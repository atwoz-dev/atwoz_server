# 소프트웨어 아키텍처 상세

## 패키지 구조

```
member/                    # 하위 도메인
├── presentation/          # API 계층
├── command/               # 명령 처리 (쓰기)
│   ├── application/       # 애플리케이션 서비스
│   ├── domain/            # 도메인 엔티티, 비즈니스 로직
│   └── infra/             # JPA 레포지토리, 외부 API
└── query/                 # 조회 전용 (읽기)
```

## 도메인 목록

- **member**: SMS 인증, 프로필 관리, 이미지 업로드, 이상형 설정, 이성 소개, 프로필 블러 해제
- **match**: 매칭 요청 메시지, 수락/거절, 메시지 내역 조회
- **heart**: 하트 보유량 조회, 하트 거래 내역 관리
- **payment**: Apple App Store 결제 연동, 영수증 검증, 하트 패키지 관리
- **notification**: FCM 기기 등록, 알림 조회/읽음/삭제, 알림 수신 설정
- **admin**: 관리자 인증, 회원 관리, 프로필 심사, 회원 정지/경고, 신고 처리
- **community**: 셀프 소개 작성/조회, 프로필 교환 신청/수락/거절
- **interview**: 설문 질문 관리, 사용자 답변 등록/수정/삭제
- **like**: 좋아요 전송, 보낸/받은 좋아요 목록 조회
- **mission**: 일일/도전 미션 진행, 미션 완료 시 하트 보상 자동 지급
- **block**: 사용자 차단
- **report**: 사용자 신고, 신고 내역 조회
- **datingexam**: 필수/선택 과목 응시, 동일 답변 기반 매칭, 소울메이트 매칭
